## 🔮 What I'm building now

### [MetaAgents](https://github.com/metaagents-ai/metaagents) — the format spec
An open standard for composing and resolving dependencies between agents, skills, and MCPs. Extends [agents.md](https://agents.md) and [Agent Skills](https://agentskills.io) with `scope`, mandatory versioning, dependency resolution, and cross-platform MCP packaging rules.

### [emploke](https://github.com/LangSensei/emploke) — the control plane
> εμπλοκή (emplokí) — entanglement: the deliberate weaving of skills, MCPs, and agents into something greater than each part.

A local-first control plane for agentic systems. Composes MetaAgents-format agents into reusable workspaces, then orchestrates them as interactive sessions or autonomous one-shot tasks. One process, one terminal, one dashboard.

```bash
npm i -g @langsensei/emploke && emploke start
```
